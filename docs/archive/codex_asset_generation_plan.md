## Runtime Conversion Rules
- Character frames: 256x256 transparent PNG
- Same scale, same foot baseline
- Facing right by default; use flipX in code
- Tiles: 128x128 PNG
- Background layers: 2048x720 PNG/WebP
- UI/VFX: transparent PNG sequences or packed sheets

## Core Gameplay Rules
- Only the two playable main characters can use Super Slap.
- NPCs and enemies cannot use Super Slap.
- Super Slap requires full meter, creates a forward shockwave, causes knockback, hit-stop, screen shake, and neon VFX.
